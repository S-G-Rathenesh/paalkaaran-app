# Install Paalkaran APK (Android)

## Steps
1. Copy `app-release.apk` to your phone.
2. On Android: Settings → Security/Privacy → **Install unknown apps** → allow your file manager/Chrome.
3. Tap the APK file to install.
4. Open the app.

## Quick test checklist
- Data saving works (milk entry saved)
- Theme & language persist after closing/reopening
- Missed date entry works
- History works
- Monthly summary works

## Notes
- Do NOT upload keystore files (`*.jks`), `key.properties`, or any passwords to GitHub.
